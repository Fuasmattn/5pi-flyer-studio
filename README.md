A single-file web app for creating on-brand concert flyers. Open `index.html` in any browser — no install, no build step.

## Features

Real-time canvas preview, 9 procedural backgrounds, 7 fonts, 4 export formats (IG Post, IG Story, FB Event, A5 Print), batch ZIP export, PDF export, QR codes, undo/redo, templates, and mobile-responsive layout.

## Live

Deployed via GitHub Pages at `https://fuasmattn.github.io/5pi-flyer-studio/`
